# KNIME Integration - Quick Start

## Overview
The CyberShield dashboard integrates with KNIME for data analysis and visualization. Charts generated by KNIME are automatically displayed in the "Security News" section.

## Quick Setup

### 1. Start the Server
```bash
node server.js
```
Server runs on `http://localhost:5500`

### 2. Data Source
KNIME reads data from: `http://localhost:5500/cases.csv`

### 3. Output Directory
KNIME should save charts to: `knime_output/` (in project root)

### 4. Chart Display
Charts are automatically loaded and displayed in the dashboard when you visit the "Security News" section.

## KNIME Workflow Steps

1. **File Reader Node**
   - URL: `http://localhost:5500/cases.csv`
   - Read header: Yes

2. **Data Processing**
   - GroupBy nodes for aggregations
   - Filter/clean data as needed

3. **Chart Creation**
   - Use Bar Chart, Pie Chart, Line Chart nodes
   - Configure axes and styling

4. **Export Charts**
   - Use Image Writer nodes
   - Save to: `knime_output/[chart_name].png`
   - Supported formats: PNG, JPG, SVG, HTML

## Supported Chart Formats

- **Images**: PNG, JPG, JPEG, SVG
- **Interactive**: HTML (displayed in iframe)

## Chart Naming

Charts are automatically titled based on filename:
- `crime_type_distribution.png` → "Crime Type Distribution"
- `status_timeline.png` → "Status Timeline"

## Troubleshooting

**No charts showing?**
- Check `knime_output/` directory exists
- Verify files are saved with correct names
- Check server is running
- Refresh browser page

**CSV not loading?**
- Ensure server.js is running
- Verify endpoint is accessible
- Check browser console for CORS errors
